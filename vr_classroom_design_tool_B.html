VR Classroom Design Tool - Travel Methods

Unity C# Oculus Rift

This project was the third assignment for UCSD's 3D User Interation class and a continuation of the previous assignment (VR Classroom Design Tool - Selection and Manipulation). This part of the project invovles implementing two different travel methods, teleportation with ray casting and continuous movement (walk mode).

Project Rubric and Directions

Goals for this Project:

  1. Implement a teleportation mode. Display a ray emanating from the right hand controller and an indicator on the floor to allow the user to point to their destination.
  2. The teleportation mode changes the user's position immediately, and their orientation stays the same.
  3. Implement a walk mode where the user is always on the floor, moves gradually, and they can only rotate around the y (up) axis.
  4. The speed of the motion in walk mode is gradually adjustable depending on the distance of the right hand's controller from its original position. The same applies to changing the user's orientation.
  5. Add a visual display to the left hand controller to indicate which mode is active.

Controls:

  • Left Hand Trigger: Toggle between teleport and walk mode
  • Right Hand Trigger: Teleport to the indicated location if in teleport mode, hold and move or turn the controller to walk around the room
