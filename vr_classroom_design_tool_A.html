VR Classroom Design Tool - Selection and Manipulation

Unity C# Oculus Rift

This project was the second assignment for UCSD's 3D User Interation class. The program is an application that helps the user design the layout of the classroom by spawning chairs and desks. The project invovles implementing object selection and manipulation using ray casting.

Project Rubric and Directions

Goals for this Project:

  1. Display a ray emanating from the user's right hand controller, pointing forward like a laser pointer.
  2. When the spawn button(A/B) is pressed, an instance of a desk or chair is created.
  3. While the user holds down a spawn button, the desk/chair remains on the ray so that it can be positioned and oriented as desired.
  4. When a spawn button is released, the desk/chair falls down and comes to rest on the floor or on another piece of furniture.
  5. Implement ray casting to select a desk/chair in the room. Highlight the selected piece of furniture.
  6. When the trigger button is released, the physics engine makes the furniture should fall down as it does when it is spawned.

Controls:

  • A: Spawn a chair at a distance from the user's hand on the ray
  • B: Spawn a desk at a distance from the user's hand on the ray
  • Right Hand Trigger: Hold to move and orient the selected furniture
