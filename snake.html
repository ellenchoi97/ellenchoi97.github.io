About

Snake

Python Pygame

This project is a remake of the classic arcade game, Snake, using the Pygame module for the graphics of the game. The game involves a single player and a two player mode. The single player mode is the traditional playing mode for the game. The two player mode involves two players competing against each other for the food, and the game ends when one player dies by going out of bounds or colliding with its body.
