MemeMaster

HTML CSS JavaScript

MemeMaster is a web application that allows the user to create, edit, share, and delete memes. This project also uses Google Firebase as its storage database.

Link to Web Application

Goals for this Project:

  1. Implement a sign-up and login mechanism to allow users to view their memes privately.
  2. Allow the user to add existing memes, either through upload or by URL.
  3. Allow the user to compose their own memes by
    • Importing an image (either through upload or by URL).
    • Adding 1-2 lines of meme style text (Impact font).
    • Include a customizable feature (changing font and its size).
  4. Allow a locallly created meme to be edited.
  5. List the memes so that the user may see what memes they have.
  6. Allow the user to delete memes. Include a confirmation of deletion before deleting the meme.
  7. Implement a distribution mechanism to allow the user to share their memes (download to local computer and copy URL to clipboard).
  8. Allow the user to manage their account by implementing logout and password change mechanisms.
