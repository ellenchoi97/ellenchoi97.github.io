Adventure Machine VR

Unity C# Oculus Rift

Adventure Machine VR is an application that allows the user to make music with a launchpad and a synthesizer inside an environment that models a bedroom. This project was the final project for UCSD's 3D User Interation class and was created in collaboration with another student.

Link to Video Demo
Project Rubric and Directions

Technical Features Implemented:

  1. Object Manipulation (grab with virtual hand)
  2. Travel (move by teleportation and move my "grabbing the air")
  3. System Control (wrist menu and sound control)
  4. Interactive Tutorial

Controls:

  • Press and hold the left or right hand trigger and have the corresponding virtual hand collide into an object to grab it.
  • "Grab the Air" travel mode: Grab the air and move the controller in the direction opposite of where you want to go.
  • Teleportation travel mode: Point the ray (on the virtual left hand) at the floor and press the left hand trigger to move to the marked spot.
  • Have the virtual hand collide with a launchpad button or synthesizer key to press it, as you would in the real world.
    • This works best when the middle finger rests on top of the hand trigger (not pressing it) so that only the virtual hand's index finger is extended.
  • Wrist Menu: Orient the left virtual hand as if looking at a watch. Press the menu items as you would with the launchpad buttons or synthesizer keys.
  • Sound Control:
    • Dials: Grab the dials and turn the virtual hand (around the y axis).
    • Slider: Grab the handle and move the virtual hand in the desired direction.
