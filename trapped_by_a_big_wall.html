Trapped By A Big Wall

Unity C# Oculus Rift

This project was the first assignment for UCSD's 3D User Interation class and was created in collaboration with another student. The purpose of this project was to get familiar with using Unity/C# and creating a VR application with an Oculus Rift headset. This project involved implementing object selection and manipulation with visual indicators and physics simulation.

Goals for this Project:

Project Rubric and Directions
  1. Create a textured plane to represent the ground.
  2. Procedurally generate a cylindrical wall of bricks. Add a texture and apply realistic physics properties to the bricks.
  3. When the user stares at a brick for at least 2 seconds, allow the user to control the brick.
    • Depending on the user's head movement, the brick will move towards the outside or the inside of the wall.
    • The brick moves parallel to the axis of the radius of the brick cylinder.
    • The physics simulation makes the bricks above a removed brick tumble down.
  4. Highlight the brick that is being looked at.
  5. Reset the wall when the user looks up to the sky.
  6. When the user stares at a brick for at least 7 seconds, explode the brick.
  7. Add a radial widget to indicate how much longer the user must stare at a brick to activate movement and explosion.
